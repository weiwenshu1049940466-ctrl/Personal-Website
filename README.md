# Personal-Website

这是一个纯静态的个人网页，可以直接打开 `index.html` 预览，也可以部署到 GitHub Pages。

## 文件结构

- `index.html`：首页内容结构
- `project-01.html`：项目详情页
- `styles.css`：页面视觉样式与响应式布局
- `script.js`：移动端导航和滚动显现动效
- `assets/`：项目图片素材
- `papers/`：论文 PDF 文件

## 使用方式

直接双击 `index.html` 即可打开。

如果你想本地起服务，可以在当前目录运行：

```bash
python3 -m http.server 4173
```
